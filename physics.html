Course Description

Physics is taught by Mr. Ellis and it is a challenging yet mentally stimulating class that teaches conceptual physics through lots and lots of problem-solving. Through end-of-chapter problems (EOCS), rigorous tests, Excel writeups, and formal problem writeups, we learn many of the tools we need to think and work like physicists. For those who have previously studied Calculus, or want to try something new, a Calculus track is offered which lines up with the AP Physics C curriculum.

Über Rocket

Every term, Mr. Ellis assigns us a physics problem to solve that has many different parts to get a final answer. With the Über Rocket problem, we were tasked with determining the total time a rocket traveled through the air with different accelerations. Once we solve the problem, we create a formal write-up in Microsoft Word.

Physics Simulation

The physics simulation was an extra credit option in which you had to create an interactive program that could model projectile motion in 2d with outputs displayed. I chose to work with Javascript along with the p5.js library for the coding and Bootstrap was used as a stylesheet.
